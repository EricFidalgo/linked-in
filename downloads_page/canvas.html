Course Content Creator using Aggregation and Operator Overloading

In this project I learned about the idea of using aggregation, operator overloading and dynamic allocation within classes.


The program has 3 classes, 2 of which are subclasses from each other. I had to make all three classes communicate with each other and then I had to use operator overloading to process and manage the results.
