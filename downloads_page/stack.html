Infix to Postfix Conversion Program

Infix to Postfix Conversion Using My Custom Stack Library in C++

In this project I learned the inner workings of how the stack data structure works by recreating the Stack standard library. I also learned how to implement this data structure into practical use by pushing and popping the operands in my program by using the Last-In-First-Out (LIFO) structure.


The program uses template declaration in the Stack class to be able to use any sort of input data type. This implementation shows my understanding in generic programming and its advantages in re-usable code.
