Postfix to Infix Conversion Program

Postfix to Infix Conversion Using a Binary Tree in C++

In this project, I learned how to create a binary tree and use it to convert a user's input from a postfix expression to an infix expression. Additionally, I gained experience in counting the number of leaves in a tree and working with more complex data structure syntax.


The program constructs a binary tree from the user's postfix input, traverses the tree to convert it into an infix expression by combining subtrees, and prints the result. It also counts and displays the total number of nodes and leaves in the tree.
