LLC Web Scraping App

A Web Scraping Tool for Florida's LLC Databases

This application grabs data from sunbiz.org and processes it into a clean, modern Excel file format. The output includes LLC ownership names and addresses and documents.


This application was developed for a real estate wholesaler who had an Excel file of LLC names but lacked information about ownership, members, and other details. This application efficiently solves that problem


Development Period: May 2024 - August 2024 (Phase 1 completed). The next phase is currently on hold.

Tools Used

  • Python 3.9
  • BeautifulSoup4
  • Selenium WebDriver
  • Pandas
  • AI Assistance
  • SQLite

Development Timeline

  • Started: May 2024
  • Completed: July 2024
  • Working on public release
